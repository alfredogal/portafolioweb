Kyphon

As part of a team of three, I developed a custom inspection machine from scratch for Medtronic, a leading medical company. The machine was able to inspect over 100 different models of medical products (biopsies) that were packaged in kits. Each kit consisted of various components that needed to be inspected for various criteria (diameter, shape, color, length, sharpness, and quantity). The machine also checked that all components of the kit were present. I used Solidworks for the design of the equipment, Cognex vision system for the inspections, and C# for the user interface and automation. I wrote Python scripts to automate the process using the Telnet port, SR232 cable, and serial port. The system was able to send ASCII data to another computer during the inspection process and allowed for supervision through a password-protected user login. This project required a combination of mechanical design, programming, and automation skills, and I am proud to have contributed to the development of a reliable and efficient inspection system for Medtronic.
